## 🧠 Terms in plain English

- **Wallet**: Your crypto account
- **Network**: The blockchain you use
- **Gas**: The fee paid to make a transaction
- **Swap**: One token traded for another
- **Mempool**: Pending transactions waiting to be added
- **EVM**: A common blockchain system used by Ethereum and many related chains
- **MEV**: A way to act on transaction order on-chain

## 📌 Troubleshooting

### The app will not open

- Make sure you downloaded the latest release
- Try opening it again as admin

### The app opens but does nothing

- Check your network choice

- Review the log area for messages
- Restart the app

### The screen looks too small

- Use Windows display scaling
- Maximize the app window
- Change your screen resolution if needed

### The bot does not start

- Check that your settings are saved
- Make sure your wallet or account is active
